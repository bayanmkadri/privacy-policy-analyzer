# Privacy Policy Analyzer

Analyze privacy policies using AI techniques and an interactive React interface.

##  Overview

Privacy Policy Analyzer is a project designed to analyze the content of privacy policies using Natural Language Processing (NLP) techniques, and classify them based on transparency and potential risks.

##  Features

- Interactive frontend built with React
- Text analysis of privacy policies using Python and NLP
- Classification of policies based on clarity and safety
- Bilingual support (Arabic / English)
- Visual and user-friendly presentation of results

##  Technologies Used

- Frontend: React, HTML, CSS, JavaScript
- Backend: Python (Flask or FastAPI)
- NLP: spaCy, Transformers, Sentiment Analysis
- Tools: Git, VS Code, GitHub

## Future Plans
- Support for uploading and analyzing PDF files
- Enhanced classification using deep learning models
- Admin dashboard for managing analysis results
